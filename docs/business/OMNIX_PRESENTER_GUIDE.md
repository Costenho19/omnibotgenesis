## PART 4 — KEY PHRASES TO USE (and avoid)

### USE these words:

- **infrastructure**
- **governance**
- **capital protection**
- **institutional**
- **auditability**
- **risk management**
- **fail-closed**
- **pre-execution validation**

### NEVER say these words:

- ~~bot~~ — makes it sound like a retail trading tool
- ~~trading system~~ — sounds like a strategy, not infrastructure
- ~~strategy~~ — OMNIX doesn't generate strategies, it governs them
- ~~algorithm~~ — too generic, too many negative associations
