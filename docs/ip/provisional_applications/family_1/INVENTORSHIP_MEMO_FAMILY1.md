## 3. SUBJECT MATTER COVERED BY THIS FAMILY

Patent Family 1 covers the following inventive concepts:

| Component | Description | Coverage |
|-----------|-------------|----------|
| **1A** | Domain-Agnostic Decision Governance Engine with Sequential Checkpoint Architecture | Core governance pipeline: CP-0 through CP-11, TIE, fail-closed default, normalized signal set, domain adapter interchangeability |
| **1B** | Counterfactual Shadow Portfolio for Automated Risk Filter Calibration | Shadow event logger, outcome tracker, counterfactual analyzer, calibration engine, human-approval requirement |
| **1C** | Decision Contradiction Index (DCI) | Pairwise divergence computation, ALIGNED/TENSIONED/CONTRADICTORY classification, mandatory HOLD on CONTRADICTORY state |

**What this family does NOT cover:**

- The Confidence-Adaptive Entry System / CAES (covered in Family 2 — OMNIX-PAT-2026-002)
- The Automated Sharia Compliance Engine (covered in Family 3 — OMNIX-PAT-2026-003)
- The Dual-Layer Trade Authorization with Voice Biometrics and PQC (covered in Family 3 — OMNIX-PAT-2026-003)
