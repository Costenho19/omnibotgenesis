⚙️ Troubleshooting
Error: "Módulo de bolsa no activado"
Solución: Verifica .env:

STOCK_TRADING_ENABLED=true

Error: "No conectado a Alpaca"
Solución: Verifica API keys en .env:

ALPACA_API_KEY=tu_key_real_aqui
ALPACA_API_SECRET=tu_secret_real_aqui

Error: "Mercado cerrado"
Solución: Espera horario de mercado (9:30am-4pm EST, Lun-Vie)

Análisis devuelve None
Solución:

Verifica que el ticker sea válido (ej: AAPL no APL)
Alpha Vantage tiene límite de 5 requests/min (modo demo)
Obtén API key gratuita: https://www.alphavantage.co/support/#api-key
